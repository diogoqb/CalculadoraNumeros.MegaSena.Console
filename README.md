# 📊 Calculadora Mega-Sena - Guia de Execução

Este aplicativo console gera estatísticas e análises da Mega-Sena,
incluindo tabelas e exportação para Excel **sem gráficos**.  
A pasta `Data` é usada tanto para leitura (`sorteio.csv`) quanto para gravação
(`analises_consolidado_dd-MM-yyyy_HH-mm.xlsx`).

---

## ⚙️ Pré-requisitos
- Windows (recomendado)  
- **Se distribuir self-contained:** não é necessário instalar .NET  
- **Se distribuir framework-dependent:** requer .NET 8 runtime instalado  
- Arquivo de sorteios (CSV, XLS ou XLSX) na pasta `Data`

---

## ▶️ Como executar

### Opção A - CMD
1. Clique duas vezes em **`run_app.cmd`**  
2. O programa será iniciado e exibirá o menu principal  
3. Certifique-se de que o arquivo `sorteio.csv` está na pasta `Data`

### Opção B - PowerShell
1. Abra o **PowerShell**  
2. Navegue até a pasta do aplicativo  
3. Execute:
   ```powershell
   .\run_app.ps1
